Trial API
---------

Use Trial methods of an SMDebug trial object. The methods are tools to
load output tensors collected by SageMaker Debugger for further analysis.

.. note::
  To use the following methods of the :class:`~smdebug.trials.trial.Trial` class,
  you must create a ``trial`` instance as guided
  in the previous topic `SMDebug Trial <smdebug.trials>`__.
